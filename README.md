# Hasher
Python firewall script that captures network traffic and writes logs to a file. It uses Scapy for packet sniffing and allows you to define simple filtering rules.


Requirements:

Run as Administrator (because you need access to all files).

Output to a CSV file.


Includes:

File location (full path)

File name

SHA-256 hash (SHA2 family)


Scan all files on the system.
